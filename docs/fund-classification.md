# 基金分类体系参考文档

## 分类层级说明

本项目采用**四级分类体系**：
- **一级分类**（7个）：按投资方向和资产类别划分
- **二级分类**（约20个）：按投资策略和风险收益特征划分
- **三级分类**（35个）：最细的分类粒度，对应天天基金的三级分类
- **六级分类**（c6属性标签）：基金的附加属性标签，如"持有期"、"定开"、"FOF"等

## 天天基金 API 字段对应

- **t0**: 一级分类代码（gp/zq/hh/fof/qdii/hb）
- **t1**: 二级分类代码
- **t2**: 三级分类名称（中文）
- **t2q**: 对应天天基金 FundGuideapi 的 `f[3]` 值，用于后端精确查询

## 属性标签说明（c6字段）

| 标签 | 含义 | 说明 |
|-----|------|------|
| 持有期 | 持有期基金 | 需要持有满一定期限才能赎回，通常费率更低 |
| 定开 | 定期开放基金 | 只有在特定开放期才能申购赎回 |
| FOF | FOF基金 | 基金中的基金，投资于其他基金 |
| 指增 | 指数增强基金 | 被动跟踪指数+主动增强策略 |
| 指数 | 被动指数基金 | 完全复制指数表现 |
| 转债 | 可转债基金 | 主要投资可转债 |
| 一级债 | 混合债券型一级 | 可投可转债，不可投股票 |
| 二级债 | 混合债券型二级 | 可投可转债和股票（通常≤20%） |
| 中长债 | 中长期纯债 | 投资中长期债券 |
| 短债 | 短期纯债 | 投资短期债券 |
| 存单 | 存单基金 | 投资银行存单 |
| 偏债 | 偏债混合 | 债券仓位>股票仓位 |
| 货币 | 货币市场基金 | 低风险流动性管理工具 |
| 价值 | 价值风格 | 偏好低估值股票 |
| 成长 | 成长风格 | 偏好高成长股票 |
| 红利 | 红利风格 | 偏好高分红股票 |
| 量化 | 量化策略 | 使用量化模型选股 |
| 消费/科技/医药 | 行业主题 | 聚焦特定行业 |
| 固收+ | 固收增强 | 债券打底+权益增强 |
| 权益 | 权益类 | 主要投资股票 |

## 数据来源

- 更新时间：2026-04-19

## 使用说明

1. **前端筛选**：用户选择一级/二级/三级分类时，通过 `t0`/`t1`/`t2` 字段过滤
2. **后端查询**：使用 `t2q` 字段精确匹配天天基金 API 返回的数据
3. **属性筛选**：使用 `c6` 标签进行二次筛选（如"只看持有期基金"）
